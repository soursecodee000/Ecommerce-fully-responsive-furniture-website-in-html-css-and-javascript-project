# Ecommerce-fully-responsive-furniture-website-in-html-css-and-javascript-project

Furniture Haven - E-commerce Website
Welcome to Furniture Haven, a fully responsive e-commerce website dedicated to selling a wide variety of furniture. This repository contains both the frontend and backend of the website, developed using PHP.

Features
Responsive Design: Furniture Haven is built with a responsive design, ensuring a seamless user experience across various devices and screen sizes.

E-commerce Functionality: The website allows users to browse through a diverse range of furniture products, view details, and make purchases online.

User Authentication: Furniture Haven provides a secure user experience with signup and login functionality, allowing users to create accounts, log in, and manage their profiles.
